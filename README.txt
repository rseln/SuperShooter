# Shooter Brawl Final Culminating Project for ICS201. Software Needed: Python 3, pygame library for the version.


About the Game:
Shooter Brawl is a multiplayer shooter brawl. You have to fight against a friend. Whoever kills the other player first wins the game. You are allowed to play as many games as
you'd like. Instructions are found within the game menu.
The game does not take long to load. You will be in the game when you click the game button.
Mouse over a button and click on it to continue to whatever menu function you'd like

Music: 
Included please turn speakers on to hear. Cannot turn off if you would like no music just turn your speakers off.

##Please make sure all the pictures and songs are in the same file as the game or else the game cannot function properly.
