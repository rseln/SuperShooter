# SuperShooter Final Culminating Project for Grade 10 ICS201. Software Needed: Python 3, pygame library for the version.

ABOUT THE GAME:
Shooter Brawl is a multiplayer shooter brawl. You have to fight against a friend. Whoever kills the other player first wins the game. You are allowed to play as many games as you'd like. Instructions are found within the game menu. The game does not take long to load. You will be in the game when you click the game button. 

MUSIC: 
Included please turn speakers on to hear.
